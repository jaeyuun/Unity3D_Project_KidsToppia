## Function
- `회원 관리` : 로그인, 회원 가입, 탈퇴
- `아바타` : 아바타 외형 변경
- `NPC` : ChatGPT 기반 안내 및 영어 회화, 아이템 구매 (IAP), 퀴즈 안내
- `길안내` : 미니맵을 통한 안내
- `커뮤니티` : 친구 추가, 삭제, 따라가기
- `퀴즈 및 도감` : 동물 및 물고기 퀴즈, 도감
- `도전과제` : 접속 일 수, 수집, 도감 진척도
- `설정` : 해상도 품질 조절, 음량 조절
